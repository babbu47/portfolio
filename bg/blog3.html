What is ReactJs?

React is a javascript front-end framework designed by facebook .It's an open source by which user can create different view application layers .

  • Reactjs works on the Model view Controller ,by the which it can handle web and mobile app in view layer.
  • React is famous for building single-page applications & mobile-apps for users-sake.

Reactjs History

Jordan Walkecreated React who worked as a software engineer in Facebook .He first released an early React project called as "FaxJS".

Features:

Reactjs Pros & cons

As like other languages ReactJs also has some of pros & cons which are following:

Pros

  • React Js is much easier to learn and use.
  • Creating Web applications become easier.
  • It offers a javascript Library.
  • It applications are also very easy to test and debug

Cons

  • It is uncomfortable for the developers to relearn new ways of doing things.
  • It only cover UI layers so you need to get a complete tooling set for development of project
  • Reactjs has poor documentation.
  • React uses Jsx by which some developers find it code hard to grasp
