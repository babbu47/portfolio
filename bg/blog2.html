What is Github?

Github is an internet hosting service for software development and others.It's an service where user can summon their different projects,contents etc.in files called as Repository.The site provide social-networking like functions such as followers and a graph to display how user work on their Repository.

Create a repository:

  • Go to Github make your first repository.You can make your repository private or public to all to showcase your projects. Repository is simply modified name of a folder that contains your projects files,image etc.

Clone A repository:

Cloning can be simply be defined as copying.You can clone a repository from Github.comor by passing the above git command in terminal to your local machine.

Cloning a repository pulls down a copy of all the repository data including all files,folder of the project.By this you can also contribute to someone repository such as by fixing an issue in other repository etc.

Pushing in Github:

An owner can make changes to it repository files by adding ,commiting and then pushing the changes to original files.WE can do all these operations by the above git commands,

Commit in github is a certified text which define about the changes the user done to its files . Pushing means the user finally want to merge its change to the original files.
